ENCODE Metrics

The ENCODE Consortium defines guidelines has established some definite guidelines for evaluating ChIP-seq samples. These include the PCR Bottlenecking Coefficient (PBC) and the Non-Redundant Fraction (NRF) for measuring library complexity in sequencing experiments.

CWL Metrics

CWL stands for Common Workflow Language, which is used for specifying analysis workflows across different environments. The Reddy Lab uses a specific CWL pipeline for processing raw ChIP-seq data, producing many useful QC metrics for validating samples such as Number of Reads In Peaks from utilizing MACS peak calling software and Number of Mapped Reads.

Fingerprint Metrics

The fingerprint analysis was developed by deepTools, a bioinformatics suite developed by the Bioinformatics Facility at the Max Planck Institute for Immunobiology and Epigenetics in Freiburg. The fingerprint analysis measures the level of enrichment versus background noise (experimental control) since ChIP-seq experiments in particular are susceptible to noise.
